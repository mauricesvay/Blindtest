SoundManager 2 Demo: “Page as playlist” UI

http://schillmania.com/projects/soundmanager2/

An example of a Muxtape.com-style UI, where an unordered list of MP3 links becomes a playlist

Flash 9 “MovieStar” edition supports MPEG4 audio as well.

Requires SoundManager 2 Javascript API.
